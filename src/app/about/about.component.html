RAFAROS-IT (Anciennement Smallwave/Bigwave-IT)

est un Site E-Commerce de vente de logiciels et de Conception-Développement d'Applications Web, Mobile(Android) et de Softwares Professionnels ou Publics.
Ce site offre également un Espace d'information sur l'actualité du monde informatique. Sa version initiale en php5/Zend/Doctrine/Pdo a évolué en Application web Java Multithread à Architecture Répartie (ci présent) réalisée sous la "norme" JAVA EE7 avec : Java 8, un Conteneur léger (Tomcat+Spring Core) et une base de données Mysql 8 ou PostgreSql - ou Oracle 12g. Sa Couche Présentation développée en Client Léger SpringMvc a finalement été mis à jour en Client Riche Angular 12.
